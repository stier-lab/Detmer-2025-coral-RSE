A. palmata Fecundity & Reproduction

Literature summary for the RSE model — Apr 2026

Current model

SC3–SC5 each produce 48,274 embryos per colony per year (Fundemar 2025: 1,255,111 embryos / 26 colonies). SC1–SC2 do not reproduce. Fecundity is size-independent.

The literature shows fecundity is strongly size-dependent. SC3 colonies are ~10% fertile; we currently treat them as 100%.

The 15% sibling-cross rate matters for restoration: outplanted populations with limited genotype diversity will have reduced reproductive output even if colonies reach spawning size.

SC5 estimate (~55,800) is consistent with the current flat rate (48,274), which is reassuring. The main correction is reducing SC3 from 48,274 to ~340 and SC4 from 48,274 to ~7,200.

Within-colony patterns

Soong & Lang (1992):

  • Upper branch surfaces are significantly more fecund than lower surfaces (P < 0.01)
  • Branch tips are infertile (3–10 cm from tip); oogenesis takes ≥6 months
  • Encrusting bases are infertile
  • Soft tissue is ~2× thicker on upper surfaces

Fragments from branch tips are disproportionately non-reproductive. This amplifies the benefit of grow-out before outplanting.

Steward (2024) IPM sub-kernel

Steward built a reproduction sub-kernel for an Acropora IPM:

r(z', z) = P_colony(z) x P_polyps(z) x f_oocytes(z) x f_recruits(z') x P_establishment

The f_oocytes(z) term was parameterized from Indo-Pacific species (Alvarez-Noriega et al. 2016) because Caribbean data were unavailable. Mendoza-Quiroz (2023) provides the Caribbean-specific values.

Recommendation

Replace the current flat fecundity rate with a size-dependent function built from A. palmata field data:

F(SC) = Area_mid  x  P_fertile  x  63.6 oocytes/cm2  x  fert_success  x  settlement_prob

Every parameter comes from published Caribbean A. palmata studies: colony fertility from Soong & Lang (1992), oocyte density from Mendoza-Quiroz et al. (2023), fertilization success from Piñón-González (2018), settlement from Fundemar. The SC5 estimate (~55,800 settlers/colony/yr) is consistent with our current Fundemar-derived value (48,274), which validates the approach. The main correction is that SC3 drops from 48,274 to ~340 and SC4 from 48,274 to ~7,200.
