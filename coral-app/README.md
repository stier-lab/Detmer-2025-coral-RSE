# Coral Restoration Model - Web Application

A production-ready, interactive web application for exploring coral restoration population dynamics using a stage-structured model for *Acropora palmata* (Elkhorn Coral).

## Features

- Interactive Model Explorer - Adjust parameters and visualize population dynamics
- Real-time Simulation - Run projections up to 100 years
- Professional UI - Modern design with smooth animations
- Accessible - WCAG 2.1 AA compliant with keyboard navigation
- Responsive - Works on desktop, tablet, and mobile devices
- Type-Safe - Built with TypeScript for reliability

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Project Created

This is a production-quality frontend application built with:
- React 18 + TypeScript
- Vite (build tool)
- TailwindCSS (styling)
- Complete model simulation engine
